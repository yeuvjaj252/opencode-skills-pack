# /csp-preview - Preview Server

$ARGUMENTS

---

## Purpose

Manage local preview lifecycle and report status.

## Typical actions

- start preview server
- stop/restart server
- check health/status
- return preview URL when available

## Suggested checks

```bash
npm run dev
```

Use framework-specific commands when the project requires a different preview start command.
