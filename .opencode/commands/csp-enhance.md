# /csp-enhance - Feature Enhancement

$ARGUMENTS

---

## Purpose

Add or improve features in an existing codebase using an iterative approach.

## Workflow

1. Understand current project state
2. Clarify enhancement scope
3. Implement in small increments
4. Validate each increment with lint/typecheck/tests
5. Summarize results and follow-up options
