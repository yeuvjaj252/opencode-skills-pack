# /csp-mobile - Mobile Development

$ARGUMENTS

---

## Focus Areas

- mobile-first UX and navigation
- performance optimization for constrained devices
- accessibility and touch ergonomics
- platform conventions (iOS/Android)

## Guidelines

- optimize render paths and avoid unnecessary re-renders
- memoize expensive components where appropriate
- keep state flow predictable
- validate behavior on multiple screen sizes
